Semi-analytical methods for plates, shells and stiffened panels
================================================================

Capabilities:
- linear buckling
- postbuckling
- Koiter asymptotic initial postbuckling analysis
- static
- vibration
- panel flutter

Installation
------------

To get the latest version::

    pip install panels
